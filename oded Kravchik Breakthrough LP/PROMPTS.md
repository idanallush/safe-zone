# פרומפטים לביצוע בקלוד קוד

## הכנה — לפני שמתחילים
1. צור תיקיית פרויקט חדשה
2. העתק את כל הקבצים מתיקיית output/ לתוך הפרויקט:
   - CLAUDE.md → root
   - context/ → context/
   - assets/ → assets/ (העתק את שני הלוגואים לכאן)

---

## פרומפט 1 — יצירת דף הנחיתה

העתק והדבק בקלוד קוד:

```
קרא את CLAUDE.md וכל הקבצים בתיקיית context/ לפני שאתה מתחיל.
קרא גם את הסיקל /mnt/skills/public/frontend-design/SKILL.md לגשת לעיצוב ברמה גבוהה.

בנה דף נחיתה שלם בקובץ HTML יחיד (index.html) עבור קורס "שוברים תקרות" של ד"ר עודד קרבצ'יק.

הדף צריך לכלול את כל הסקשנים שמתוארים ב-CLAUDE.md, לפי הסדר.

דגשים קריטיים:
1. הקופי צריך להיות בעברית, ישיר, חם, לא גנרי. עקוב אחרי copy-brief.md בדיוק.
2. העיצוב צריך להיות כהה, מודרני, פרימיום עם נגיעות זהב. עקוב אחרי design-brief.md בדיוק.
3. RTL מלא, מובייל-פירסט, רספונסיבי.
4. YouTube embeds עם lazy loading.
5. הטופס שולח POST ל-placeholder (אשלים endpoint בהמשך).
6. הלוגו הלבן (logo-white.png) ב-hero. תמיר אותו ל-base64 ותטמיע inline.
7. Smooth scroll לכפתורי CTA.
8. אין תלויות חיצוניות מלבד Google Fonts (Heebo).

תתחיל לבנות. קובץ HTML אחד, הכל self-contained.
```

## פרומפט 2 — סקירה ותיקונים (אחרי שהדף נבנה)

```
פתח את index.html ותעשה review מלא:

1. קופי:
   - האם יש משפטים גנריים/קלישאות? תחליף אותם.
   - האם הטון עקבי — ישיר, חם, לא שיווקי-מאגר?
   - האם יש שגיאות כתיב/דקדוק בעברית?
   - האם כל סקשן עומד בפני עצמו (גולש שקופץ לאמצע מבין)?

2. עיצוב:
   - האם הצבעים עקביים? (שחור #0D0D0D, זהב #D4A843, לבן #FFFFFF)
   - האם יש מספיק whitespace בין סקשנים?
   - האם הכפתורים בולטים ומזמינים ללחיצה?
   - האם הדף נראה טוב במובייל (375px)?
   - האם הדף נראה טוב בדסקטופ (1440px)?

3. טכני:
   - האם הטופס עובד? (שדות required, validation בסיסי)
   - האם ה-YouTube embeds עם lazy loading?
   - האם smooth scroll עובד?
   - האם אין console errors?

תקן כל מה שמצאת.
```

## פרומפט 3 — חיבור lead.im (אחרי שתקבל endpoint)

```
חבר את הטופס בדף ל-lead.im.

Endpoint: [כאן תשלים את ה-URL שתקבל מעודד]
Method: POST
Content-Type: application/json

שדות:
- name: שם מלא
- phone: טלפון
- email: אימייל

אחרי שליחה מוצלחת:
- הסתר את הטופס
- הצג הודעת תודה: "תודה! עודד יחזור אליך בהקדם."
- הודעת שגיאה: "משהו השתבש, נסה שוב או התקשר ל-054-8362202"

אל תשנה שום דבר אחר בדף.
```

## פרומפט 4 — דיפלוי (אופציונלי — אם רוצים Cloudflare Pages)

```
הכן את הפרויקט לדיפלוי ב-Cloudflare Pages:
1. ודא שכל הקבצים (HTML + assets) בתיקייה אחת
2. צור .gitignore בסיסי
3. ודא שהלוגו embedded כ-base64 בתוך ה-HTML (כדי שהכל self-contained)
4. ודא שאין תלויות חיצוניות שדורשות build step

הפרויקט צריך להיות deploy-ready — פשוט להעלות את התיקייה ל-Cloudflare Pages.
```
