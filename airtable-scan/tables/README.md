# tables/

## מה לשים פה
לכל טבלה ב-Airtable שלוש פריטים:

### 1. CSV Export
- ב-Airtable: פתח את הטבלה > `...` (תפריט) > Download CSV
- שמור בשם: `[שם-הטבלה].csv`
- דוגמה: `הוצאות.csv`, `הכנסות.csv`, `קטגוריות.csv`

### 2. צילום מסך של השדות
- ב-Airtable: לחץ על `Fields` בסרגל הצד
- צלם מסך שרואים את כל השדות + הטיפוסים (Text, Number, Formula, Linked Record...)
- שמור בשם: `[שם-הטבלה]-fields.png`
- **חשוב**: וודא שרואים את סוג השדה ליד כל שם

### 3. נוסחאות (אם יש)
- לכל שדה מסוג Formula / Rollup / Lookup: לחץ עליו וצלם את הנוסחה
- שמור בשם: `[שם-הטבלה]-formulas.png`
- אם יש הרבה נוסחאות, אפשר כמה צילומים: `הוצאות-formula-1.png`, `הוצאות-formula-2.png`

## למה זה חשוב
- ה-CSV נותן את הנתונים עצמם ואת מבנה העמודות
- צילום השדות נותן את הטיפוסים, ה-linked records, וה-constraints
- הנוסחאות נותנות את הלוגיקה העסקית שרצה מאחורי הקלעים
