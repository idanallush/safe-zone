# automations/

## מה לשים פה
לכל Automation ב-Airtable:

### 1. צילום כללי
- פתח את ה-Automation
- צלם מסך שרואים את כל ה-flow (trigger + actions)
- שמור בשם: `[שם-האוטומציה]-overview.png`

### 2. צילום ה-Trigger
- לחץ על ה-Trigger
- צלם את ההגדרות (איזה טבלה, איזה תנאי, איזה schedule)
- שמור בשם: `[שם]-trigger.png`

### 3. צילום כל Action
- לחץ על כל step
- צלם את ההגדרות (איזה טבלה, איזה שדות, איזה ערכים)
- שמור בשם: `[שם]-step-1.png`, `[שם]-step-2.png`

## למה זה חשוב
אוטומציות = הלוגיקה העסקית שרצה ברקע. בלי זה קלוד קוד יבנה ממשק יפה אבל בלי ה-backend logic.
