# interfaces/

## מה לשים פה
לכל Interface ב-Airtable שני צילומי מסך:

### 1. מצב צפייה (View Mode)
- פתח את ה-Interface כרגיל
- צלם מסך מלא
- שמור בשם: `[שם]-view.png`
- דוגמה: `expenses-overview-view.png`

### 2. מצב עריכה (Edit Mode)
- לחץ על Edit interface (אייקון העיפרון)
- צלם כל אלמנט / section — כולל ה-settings שלו (data source, filters)
- שמור בשם: `[שם]-edit-1.png`, `[שם]-edit-2.png`

### 3. אופציונלי: לחיצה על אלמנט ספציפי
- אם יש אלמנט מורכב (chart, filtered list, summary), לחץ עליו במצב עריכה
- צלם את ההגדרות שנפתחות (filters, grouping, fields)
- שמור בשם: `[שם]-element-[תיאור].png`

## למה זה חשוב
- מצב צפייה = UX שקלוד קוד צריך לשחזר
- מצב עריכה = הלוגיקה מאחורי כל אלמנט (מאיפה הדאטה, איזה פילטרים)
