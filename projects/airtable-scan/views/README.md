# views/

## מה לשים פה
Views עם פילטרים, מיונים, או groups שמשפיעים על איך הנתונים מוצגים.

### מה לצלם
- פתח את ה-View
- לחץ על Filter / Group / Sort כדי לפתוח את ההגדרות
- צלם מסך עם ההגדרות פתוחות
- שמור בשם: `[שם-טבלה]-[שם-view].png`

### דוגמאות
- `הוצאות-monthly-view.png` — תצוגה חודשית עם פילטר תאריכים
- `הוצאות-by-category.png` — תצוגה מקובצת לפי קטגוריה

## מתי לא צריך
אם ה-View הוא פשוט Grid רגיל בלי פילטרים מיוחדים, לא צריך לצלם אותו. ה-CSV כבר מכסה את זה.
